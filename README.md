# MoodTracker

![mt](https://user-images.githubusercontent.com/54987308/170805647-f1acf72c-f286-4a15-bd5f-7e03effaa5b1.gif)

Tech Used : 
  Dagger2
  Coroutines
  Navigation component
  Room Db
  Jetpack Android
